Use Case Description

Students and professionals often struggle to find the right courses from the overwhelming number of options available online. This AI assistant solves that by acting as a personalised learning advisor — it holds a natural multi-turn conversation with the user, progressively collecting information about their interest domain, current skill level, specific goal, and weekly time commitment.

Once enough context is gathered, it generates a structured response containing:

  • Top 3 Course Recommendations — with platform, duration, and justification
  • 6-Month Learning Roadmap — month-by-month milestones
  • Study Tips — personalised to the learner's situation
  • Next Steps — career moves and certifications after completion

Unlike a static FAQ bot, this assistant maintains full conversation history across turns, enabling follow-up questions, refinements, and deeper exploration — making it a genuinely useful real-world AI product.

Implementation Steps
  • Bot sends a warm welcome message with 6 quick-start suggestion chips covering popular domains.
  • User selects or types their interest domain (e.g., Python, AI, Web Dev, Cloud).
  • Bot asks for the user's current skill level (Beginner / Intermediate / Advanced).
  • Bot asks for their specific learning goal (get a job, build a project, hobby, etc.).
  • Bot asks for weekly time commitment (less than 5 hrs / 5–10 / 10–20 / 20+).
  • Full conversation history is sent to Gemini 1.5 Flash API with an expert system prompt.
  • Structured recommendations returned — top 3 courses, 6-month roadmap, tips, and next steps. User can ask follow-up questions freely.
